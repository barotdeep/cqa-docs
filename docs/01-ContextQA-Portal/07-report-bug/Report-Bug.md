# Bug report

This guide provides a quick and straightforward process for submitting bug reports on the ContextQA Dashboard. It outlines the steps to navigate to the bug report section, add the issues, and submit the report. Following this guide will ensure that any bugs or issues are effectively communicated and addressed by the relevant team.

### Steps

1\. Navigate to **ContextQA Dashboard**

2\. Click on **Bug Report**

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2024-03-19/34eff4f3-9b75-46e3-94ad-0403fe90f736/ascreenshot.jpeg?tl_px=0,113&br_px=967,654&force_format=png&width=967&wat_scale=86&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=5,239)
